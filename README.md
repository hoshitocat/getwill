# リタリコ2dayハッカソン
## チーム：りこりこ
## サービス名：getwill
### 概要
自分の住んでいる街をみんなで良くしていくためのサービス

街が変われば、地域が変わる。

地域が変われば、国が変わる。

国が変われば、世界が変わる。

### サービス付加価値
* 意志の重要度を可視化
* 行政と個人のプラットフォームを統一（2chと行政の合体）
